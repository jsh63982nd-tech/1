# 발송배전기술사 기출회독

개인용 발송배전기술사 기출 회독 웹앱입니다.

## 실행 방법

1. `start-app.bat`을 실행합니다.
2. 브라우저에서 `http://localhost:5173`으로 접속합니다.

서버 없이 간단히 확인하려면 `index.html`을 직접 열어도 됩니다.

## 포함 데이터

- 104회, 107회, 108회, 110회, 111회, 112회, 113회, 114회, 115회, 117회, 118회, 119회, 120회
- 121회부터 137회까지
- 총 930문제

101회와 102회 PDF는 이미지 스캔본이라 OCR 처리가 필요해서 현재 데이터에는 포함되어 있지 않습니다.
